# Personal Website

The repo for my own website: http://officialtbb.space/ \
Also this project was mainly generated by Cursor with Claude 3.5, I am using it as my learning to become a fullstack AI era programmer.

## Tech Stack

### Frontend
- React 18
- Material-UI
- React Router
- Styled Components
- Axios for API calls

### Backend
- Spring Boot 3
- Spring Security
- Spring Data JPA
- H2 Database
- JWT Authentication
- Maven

## Prerequisites

- Node.js 16+ and npm
- Java 17+
- Maven 3.6+
